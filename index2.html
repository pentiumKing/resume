Types of Lists

There are two main types of Lists

  1. Ordered List
  2. Unodered List

Ordered list typically rendered a number list; example of an Ordered list is as follows

  1. Mix flour, baking powder, sugar, and salt.
  2. In another bowl, mix eggs, milk, and oil.
  3. Stir both mixitures together.
  4. Fill muffin tray 3/4 full.
  5. Bake for 20 minutes.

Unordered list typically rendered as a bulleted list

  • Milk
  • Cheese
    • Blue cheese
    • feta

Nesting a list

There are three ways to Nesting a list

  1. Ordered list inside Unodered list
  2. Unodered inside Unodered list
  3. combinatory list

example of Ordered list inside Unodered list

  • first item
    1. first item first subitem
    2. first item second subitem
    3. first item third subitem
  • Second item

example of Unodered list inside Unodered list

  • first item
    • first item first subitem
    • first item second subitem
    • first item third subitem
  • second item

example of combinatory list
